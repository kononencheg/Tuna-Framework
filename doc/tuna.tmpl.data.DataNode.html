Узел парсинга объекта данных передаваемого шаблону.

Аргументы конструктора:

Имя Тип Описание
value * Данные узла.
opt_parent tuna.tmpl.data.DataNode Родительский узел данных.
opt_key string Ключ узла данных в родительском.

Методы

getKey:tuna.tmpl.data.DataNode

Получение узла-ключа данных текущего узла.

Возвращает:

  • Узел-ключ данных текущего узла.

getParent:tuna.tmpl.data.DataNode

Родительский узел данных.

Возвращает:

getStringValue:string

Получение строкового значения узла данных.

Возвращает:

  • Строковое значение узла данных.

getValue:*

Получение значений узла данных.

Возвращает:

  • Значение узла данных.

growChild:tuna.tmpl.data.DataNode

Создание и возвращение дочернего узда данных по ключу.

Аргументы:

Имя Тип Описание
key string Ключ дочернего узла.

Возвращает:

  • Новый узел данных, tuna.tmpl.data.NULL_NODE в случае неудачи.

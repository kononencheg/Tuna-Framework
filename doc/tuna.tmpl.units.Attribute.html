Элемент шаблона трансформации устанавливающий значение определенного аттрибута в зависимости от данных.

Аргументы конструктора:

Имя Тип Описание
root tuna.tmpl.units.Template Корневой элемент трансформации.
attributeName string Корневой элемент трансформации.

Реализует и наследует:

  • tuna.tmpl.units.Spot

Свойства

_nodes:Array.<!Node>

[protected] Набор DOM-элементов для отображения данных.

_pattern:Array.<string>

[protected] Разбитая строка образца отображения данных.

NAME:string

Методы

[protected] _applyValue:

Отображение значения в целевых DOM-узлах.

Аргументы:

Имя Тип Описание
value * Занчение для отображения.

addTargets:

Добавление целевых DOM-элементов для отображения данных.

Аргументы:

Имя Тип Описание
elements Node | !Array.<!Node> Массив DOM-элементов.

applyData:

Обработка данных элементом шаблонизатора. Данные передаются в виде экземпляра узла дерева данных.

Аргументы:

Имя Тип Описание
dataNode tuna.tmpl.data.DataNode Узел дерева данных.

destroy:

Уничтожение функционала элемента шаблонизатора.

setEvent:

Устанока флага наличия события генерирующегося у элемента при изменении аттрибута.

Аргументы:

Имя Тип Описание
hasEvent boolean Флаг наличия события.

setPath:

Установка пути извлечения данных для отображения.

Аргументы:

Имя Тип Описание
path string Строка пути извлечения данных.

См. также:

  • tuna.tmpl.data.PathEvaluator

setPattern:

Установка образца отображения данных. Образец представляет собой разбитую строку разрывы которой предназначены для вставки данных.

Аргументы:

Имя Тип Описание
pattern Array.<string> Разбитая строка образца.
